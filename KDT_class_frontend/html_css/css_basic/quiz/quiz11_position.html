Lizard

Lizard is the common name used for all squamate reptiles other than snakes
(and to a lesser extent amphisbaenians),
encompassing over 7,000 species,[1] ranging across all continents except Antarctica,
as well as most oceanic island chains.
The grouping is paraphyletic as some lizards are more closely
related to snakes than they are to other lizards.
Lizards range in size from chameleons and geckos
a few centimeters long to the 3-meter-long Komodo dragon.
