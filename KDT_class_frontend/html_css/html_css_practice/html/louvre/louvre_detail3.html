민중을 이끄는 자유의 여신

「민중을 이끄는 자유의 여신」은 프랑스 화가 외젠 들라크루아가
1830년 7월 혁명을 기념하기 위해 그린 대표적인 낭만주의 회화입니다.
작품 중심에는 프랑스 국기를 든 여성 자유의 여신 ‘마리안’이
민중과 함께 바리케이드를 넘어 전진하는 장면이 담겨 있습니다.
이 여신은 자유와 저항의 상징으로, 당시 혁명 정신을 강렬하게 표현하고 있으며,
무기와 시체가 흩어진 혼란스러운 전장을 통해 현실과 이상 사이의 긴장감을 전합니다.
회화 속 인물들은 실제 시민, 노동자, 지식인 등 다양한 계층으로 묘사되어
당시 사회의 목소리를 생생히 전달합니다.
현재는 루브르 박물관에 소장되어 있으며, 프랑스 혁명 정신을 상징하는 대표적인 작품입니다.
